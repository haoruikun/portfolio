SummonerSearch

SummonerSearch is a summoner search engine that allows League of Legends players to search for summoner’s(gamers) names and get their info like ranked, wins, loss, etc. It also has a compare function for users to compare two summoners.

Core Features:

  1. Search for summoner's ranking information.
  2. Compare two summoners.

Technologies Used: HTML, CSS, JavasScript, PHP, Bootstrap, JSON, AJAX, and Riot API.
